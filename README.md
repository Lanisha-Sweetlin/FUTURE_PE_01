 AI Website Copy Generator for Local Businesses

Future Interns – Prompt Engineering Task 1 (2026)

 Project Overview

This project demonstrates how Prompt Engineering can be used to generate high-quality website copy for local businesses using AI tools.

Many small businesses struggle with writing effective website content. Their websites often lack:

* clear value propositions
* persuasive messaging
* strong call-to-action sections

Using structured prompts, this system generates conversion-focused website copy that can be used directly on business websites.

 Business Chosen

Black Door Cafe
Location: Coimbatore, Tamil Nadu
Business Type: Cafe

Black Door Cafe is a popular hangout spot known for its cozy interiors, specialty coffee, desserts, and relaxed atmosphere for students and young professionals.

---

Objective

The goal of this project is to build a prompt framework that can:

* Generate homepage copy with a strong value proposition
* Create service or menu descriptions
* Write persuasive call-to-action sections
* Adapt tone for different business types
* Produce website-ready content

Prompt Workflow

The system uses three structured prompts:

master_prompt.txt
Generates the initial website copy.

refinement_prompt.txt
Improves clarity and persuasion.

tone_modifier.txt
Adjusts tone depending on the business type.
